Tema 1: Propiedades de Texto

Propiedad Valor Inicial Descripción
color black Define el color del texto de un elemento.
font-family serif Especifica la fuente del texto.
font-size medium Establece el tamaño de la fuente.
font-weight normal Controla el grosor de la fuente (normal, bold, etc.).
font-style normal Indica el estilo de la fuente (normal, italic, oblique).
font-variant normal Controla la variante de la fuente (normal, small-caps).
font inherit Atajo para definir varias propiedades de fuente en una sola declaración.
text-align left Establece la alineación del texto (left, right, center, justify).
line-height normal Define la altura de la línea.
text-decoration none Aplica decoraciones al texto (underline, overline, line-through).
text-transform none Controla la capitalización del texto (uppercase, lowercase, capitalize).
vertical-align baseline Alinea verticalmente el contenido (top, middle, bottom).
text-indent 0 Controla la sangría de la primera línea de un párrafo.
letter-spacing normal Ajusta el espacio entre letras.
word-spacing normal Aumenta o disminuye el espacio entre palabras.
white-space normal Controla cómo se maneja el espacio en blanco (normal, nowrap, pre).

Tema 2: Enlaces

Característica Propiedades Descripción
Tamaño, Color y Decoración color, text-decoration, font-size Define el color, decoración (subrayado, etc.) y tamaño del texto de un enlace.
Decoración Personalizada background-color, border, border-radius Permite aplicar estilos personalizados al fondo y bordes de un enlace.

Tema 3: Imágenes

Característica Propiedades Descripción
Anchura y Altura width, height Define el ancho y la altura de una imagen o elemento.
Eliminar el Borde border Controla el borde alrededor de una imagen, permitiendo establecerlo a none para eliminarlo.

Tema 4: Listas

Propiedad Valor Inicial Descripción
list-style-type disc Especifica el tipo de marcador para listas desordenadas.
list-style-position outside Define la posición del marcador, ya sea fuera o dentro del bloque de texto.
list-style-image none Permite utilizar una imagen como marcador en lugar de los tipos predeterminados.
list-style none Atajo que establece todas las propiedades de estilo de lista a la vez.

Tema 5: Tablas

Propiedad Se Aplica a Valor Inicial Descripción
border-collapse table separate Controla cómo se manejan los bordes de las celdas, ya sea separados o colapsados.
border-spacing table 0 Define el espacio entre los bordes de las celdas en tablas.
empty-cells table show Controla la visibilidad de las celdas vacías, ya sea mostrarlas o ocultarlas.
caption-side table top Define la posición del título de la tabla (top o bottom).
``` Puedes copiar y pegar este código en un archivo `.html` y abrirlo en tu navegador para ver la tabla con la información organizada. Si necesitas más cambios o adiciones, ¡háznoslo saber!
